# 统计功能模块

## 一、功能概述

统计功能模块为用户提供数据洞察，帮助了解习惯养成情况：

- **周统计**：本周每日任务完成情况
- **月统计**：本月任务完成日历视图
- **年统计**：全年趋势分析
- **任务完成率**：总次数、当前连续、最长连续

## 三、周统计

### 3.1 接口说明

```
GET /stats/week?date=2026-06-09
```

**响应：**
```json
{
    "code": 200,
    "msg": "success",
    "data": [
        {
            "task_id": 1,
            "task_name": "每日跑步",
            "repeat_type": 1,
            "week_rule": "1,3,5",
            "target_count": 3,
            "week_days": [true, false, true, false, true, false, false]
        }
    ]
}
```

**week_days 说明：**
- 索引 0 = 周一，索引 6 = 周日
- true = 已打卡，false = 未打卡

### 3.2 核心算法

```javascript
router.get('/week', async (ctx) => {
    const { date } = ctx.query;
    const targetDate = new Date(date);
    
    // 1. 计算本周起止日期（周一到周日）
    const dayOfWeek = targetDate.getDay();
    const daysToMonday = dayOfWeek === 0 ? 6 : dayOfWeek - 1;
    const startDate = new Date(targetDate);
    startDate.setDate(targetDate.getDate() - daysToMonday);
    const endDate = new Date(startDate);
    endDate.setDate(startDate.getDate() + 6);
    
    // 2. 查询所有活跃任务
    const tasksSql = `SELECT id, task_name, repeat_type, week_rule, target_count
                      FROM task WHERE status = 1`;
    const [tasks] = await pool.query(tasksSql);
    
    // 3. 查询本周所有打卡记录
    const recordsSql = `
        SELECT r.task_id, r.record_date
        FROM record r
        JOIN task t ON r.task_id = t.id
        WHERE t.status = 1 AND r.record_date >= ? AND r.record_date <= ?
    `;
    const [records] = await pool.query(recordsSql, [
        formatDate(startDate), 
        formatDate(endDate)
    ]);
    
    // 4. 构建打卡记录 Map：taskId -> Set<date>
    const recordsMap = new Map();
    for (const record of records) {
        const taskId = record.task_id;
        const dateStr = formatDate(record.record_date);
        if (!recordsMap.has(taskId)) {
            recordsMap.set(taskId, new Set());
        }
        recordsMap.get(taskId).add(dateStr);
    }
    
    // 5. 生成本周七天的日期数组
    const weekDates = [];
    for (let i = 0; i < 7; i++) {
        const d = new Date(startDate);
        d.setDate(startDate.getDate() + i);
        weekDates.push(formatDate(d));
    }
    // weekDates = ["2026-06-09", "2026-06-10", ..., "2026-06-15"]
    
    // 6. 对每个任务，检查每天是否打卡
    const result = tasks.map(task => {
        const checkedInDates = recordsMap.get(task.id) || new Set();
        const weekDays = weekDates.map(d => checkedInDates.has(d));
        // weekDays = [true, false, true, false, true, false, false]
        
        return {
            task_id: task.id,
            task_name: task.task_name,
            repeat_type: task.repeat_type,
            week_rule: task.week_rule || '',
            target_count: task.target_count || 0,
            week_days: weekDays
        };
    });
    
    ctx.body = { code: 200, msg: 'success', data: result };
});
```

### 3.3 周计算原理

```
假设 date = 2026-06-11（周三）

dayOfWeek = 3 (JavaScript: 0=周日, 1=周一, ..., 3=周三)
daysToMonday = 3 - 1 = 2

startDate = 2026-06-11 - 2天 = 2026-06-09（周一）
endDate = 2026-06-09 + 6天 = 2026-06-15（周日）

本周日期：
周一 2026-06-09 → weekDays[0]
周二 2026-06-10 → weekDays[1]
周三 2026-06-11 → weekDays[2]
周四 2026-06-12 → weekDays[3]
周五 2026-06-13 → weekDays[4]
周六 2026-06-14 → weekDays[5]
周日 2026-06-15 → weekDays[6]
```

**注意：** JavaScript 的 getDay() 返回 0-6，其中 0=周日。

## 四、月统计

### 4.1 接口说明

```
GET /stats/month?date=2026-06-09
```

**响应：**
```json
{
    "code": 200,
    "msg": "success",
    "data": [
        {
            "task_id": 1,
            "task_name": "每日跑步",
            "repeat_type": 1,
            "week_rule": "1,3,5",
            "create_time": "2026-06-01",
            "month_days": [1, 3, 5, 7, 9, 11]
        }
    ]
}
```

**month_days 说明：** 本月打卡的日期列表（1-31）

### 4.2 核心算法

```javascript
router.get('/month', async (ctx) => {
    const { date } = ctx.query;
    const targetDate = new Date(date);
    const year = targetDate.getFullYear();
    const month = targetDate.getMonth();
    
    // 1. 计算本月起止日期
    const startDate = new Date(year, month, 1);           // 6月1日
    const endDate = new Date(year, month + 1, 0);         // 6月30日
    
    // 2. 查询任务
    const [tasks] = await pool.query(tasksSql);
    
    // 3. 查询本月打卡记录
    const [records] = await pool.query(recordsSql, [
        formatDate(startDate), 
        formatDate(endDate)
    ]);
    
    // 4. 构建 Map：taskId -> Set<day>
    const recordsMap = new Map();
    for (const record of records) {
        const taskId = record.task_id;
        const day = record.record_date.getDate();  // 1-31
        if (!recordsMap.has(taskId)) {
            recordsMap.set(taskId, new Set());
        }
        recordsMap.get(taskId).add(day);
    }
    
    // 5. 组装结果
    const result = tasks.map(task => {
        const checkedInDays = recordsMap.get(task.id) || new Set();
        const monthDays = Array.from(checkedInDays).sort((a, b) => a - b);
        // monthDays = [1, 3, 5, 7, 9, 11]
        
        return {
            task_id: task.id,
            task_name: task.task_name,
            repeat_type: task.repeat_type,
            week_rule: task.week_rule || '',
            create_time: task.create_time ? formatDate(task.create_time) : null,
            month_days: monthDays
        };
    });
    
    ctx.body = { code: 200, msg: 'success', data: result };
});
```

## 五、年统计

### 5.1 接口说明

```
GET /stats/year?year=2026
```

**响应：**
```json
{
    "code": 200,
    "msg": "success",
    "data": [
        {
            "task_id": 1,
            "task_name": "每日跑步",
            "total_days": 45,
            "months": [
                { "month": 1, "days": [1, 3, 5, 7, ...] },
                { "month": 2, "days": [1, 3, 5, ...] },
                { "month": 3, "days": [1, 3, 5, ...] }
            ]
        }
    ]
}
```

### 5.2 核心算法

```javascript
router.get('/year', async (ctx) => {
    const { year } = ctx.query;
    const yearNum = parseInt(year);
    
    // 1. 年范围
    const startDateStr = `${yearNum}-01-01`;
    const endDateStr = `${yearNum}-12-31`;
    
    // 2. 查询任务和记录
    const [tasks] = await pool.query(tasksSql);
    const [records] = await pool.query(recordsSql, [startDateStr, endDateStr]);
    
    // 3. 构建双层 Map：taskId -> (month -> Set<day>)
    const recordsMap = new Map();
    for (const record of records) {
        const taskId = record.task_id;
        const month = record.record_date.getMonth() + 1;  // 1-12
        const day = record.record_date.getDate();
        
        if (!recordsMap.has(taskId)) {
            recordsMap.set(taskId, new Map());
        }
        const monthMap = recordsMap.get(taskId);
        
        if (!monthMap.has(month)) {
            monthMap.set(month, new Set());
        }
        monthMap.get(month).add(day);
    }
    
    // 4. 组装结果
    const result = tasks.map(task => {
        const monthMap = recordsMap.get(task.id) || new Map();
        const months = [];
        let totalDays = 0;
        
        // 遍历 1-12 月
        for (let m = 1; m <= 12; m++) {
            const days = monthMap.get(m) || new Set();
            if (days.size > 0) {
                months.push({
                    month: m,
                    days: Array.from(days).sort((a, b) => a - b)
                });
                totalDays += days.size;
            }
        }
        
        return {
            task_id: task.id,
            task_name: task.task_name,
            total_days: totalDays,
            months: months
        };
    });
    
    ctx.body = { code: 200, msg: 'success', data: result };
});
```

## 六、任务完成率

### 6.1 接口说明

```
GET /stats/rate/1
```

**响应：**
```json
{
    "code": 200,
    "msg": "success",
    "data": {
        "total_count": 45,
        "current_streak": 5,
        "max_streak": 12
    }
}
```

### 6.2 核心算法

```javascript
router.get('/rate/:taskId', async (ctx) => {
    const { taskId } = ctx.params;
    
    // 1. 检查任务是否存在
    const checkTaskSql = `SELECT * FROM task WHERE id = ?`;
    const [tasks] = await pool.query(checkTaskSql, [taskId]);
    if (tasks.length === 0) {
        ctx.body = { code: 404, msg: '任务不存在' };
        return;
    }
    
    // 2. 总打卡次数
    const totalCountSql = `SELECT COUNT(*) as count FROM record WHERE task_id = ?`;
    const [totalCountResult] = await pool.query(totalCountSql, [taskId]);
    const totalCount = totalCountResult[0].count;
    
    // 3. 获取所有打卡日期（排序）
    const allRecordsSql = `
        SELECT record_date FROM record 
        WHERE task_id = ? ORDER BY record_date ASC
    `;
    const [allRecords] = await pool.query(allRecordsSql, [taskId]);
    
    const dates = allRecords.map(r => new Date(formatDate(r.record_date)));
    // dates = [2026-06-01, 2026-06-02, 2026-06-03, ...]
    
    let currentStreak = 0;
    let maxStreak = 0;
    
    // 4. 计算当前连续天数
    if (dates.length > 0) {
        const sortedDates = [...dates].sort((a, b) => a - b);
        const lastDate = sortedDates[sortedDates.length - 1];
        
        const today = new Date();
        today.setHours(0, 0, 0, 0);
        
        // 最后打卡日期距离今天 <= 1 天
        const daysDiff = Math.floor((today - lastDate) / (1000 * 60 * 60 * 24));
        
        if (daysDiff <= 1) {
            currentStreak = 1;
            
            // 从后往前数连续
            for (let i = sortedDates.length - 2; i >= 0; i--) {
                const diff = Math.floor(
                    (sortedDates[i + 1] - sortedDates[i]) / (1000 * 60 * 60 * 24)
                );
                if (diff === 1) {
                    currentStreak++;
                } else {
                    break;  // 不连续，停止
                }
            }
        }
    }
    
    // 5. 计算历史最长连续
    let tempStreak = 1;
    for (let i = 1; i < sortedDates.length; i++) {
        const diff = Math.floor(
            (sortedDates[i] - sortedDates[i - 1]) / (1000 * 60 * 60 * 24)
        );
        if (diff === 1) {
            tempStreak++;
        } else {
            maxStreak = Math.max(maxStreak, tempStreak);
            tempStreak = 1;
        }
    }
    maxStreak = Math.max(maxStreak, tempStreak);
    
    ctx.body = {
        code: 200,
        msg: 'success',
        data: {
            total_count: totalCount,
            current_streak: currentStreak,
            max_streak: maxStreak
        }
    };
});
```

### 6.3 连续天数计算示例

```
打卡日期：6月1日, 6月2日, 6月3日, 6月5日, 6月6日

排序后：[06-01, 06-02, 06-03, 06-05, 06-06]

计算最长连续：
i=1: 06-02 - 06-01 = 1天 → tempStreak = 2
i=2: 06-03 - 06-02 = 1天 → tempStreak = 3
i=3: 06-05 - 06-03 = 2天 → 中断！
     maxStreak = max(0, 3) = 3
     tempStreak = 1
i=4: 06-06 - 06-05 = 1天 → tempStreak = 2

循环结束：
maxStreak = max(3, 2) = 3

当前连续（假设今天是 6月7日）：
最后打卡是 6月6日，距离今天 1 天
从后往前数：
06-06 到 06-05 = 1天 → currentStreak = 2
06-05 到 06-03 = 2天 → 中断

结果：current_streak = 2, max_streak = 3
```

### 7.2 周统计页面

```kotlin
class WeekStatsFragment : Fragment(R.layout.fragment_week_stats) {
    private var currentDate: LocalDate = LocalDate.now()
    private val today: LocalDate = LocalDate.now()
    
    override fun onViewCreated(view: View, savedInstanceState: Bundle?) {
        val recyclerView = view.findViewById<RecyclerView>(R.id.rvWeekStat)
        val tvWeekRange = view.findViewById<TextView>(R.id.tv_week_range)
        
        // 显示周范围：06/09 ~ 06/15
        updateWeekRange(tvWeekRange)
        
        // 加载数据
        loadWeekData(currentDate, recyclerView)
    }
    
    private fun updateWeekRange(tv: TextView) {
        val startOfWeek = currentDate.with(DayOfWeek.MONDAY)
        val endOfWeek = currentDate.with(DayOfWeek.SUNDAY)
        val formatter = DateTimeFormatter.ofPattern("MM/dd")
        tv.text = "${startOfWeek.format(formatter)} ~ ${endOfWeek.format(formatter)}"
    }
    
    private fun loadWeekData(date: LocalDate, recyclerView: RecyclerView) {
        val dateStr = date.format(DateTimeFormatter.ISO_LOCAL_DATE)
        
        lifecycleScope.launch {
            val response = apiService.getWeekStats(dateStr)
            if (response.isSuccessful && response.body()?.code == 200) {
                val weekStats = response.body()?.data ?: emptyList()
                
                // 计算每周目标
                val data = weekStats.map { stat ->
                    WeekTaskStat(
                        taskId = stat.task_id,
                        taskName = stat.task_name,
                        weekStatus = stat.week_days,
                        targetRequired = calculateWeeklyTarget(
                            stat.repeat_type, 
                            stat.week_rule, 
                            stat.target_count
                        )
                    )
                }
                
                recyclerView.adapter = WeekStatAdapter(data)
            }
        }
    }
    
    // 根据重复类型计算每周目标
    private fun calculateWeeklyTarget(repeatType: Int, weekRule: String, targetCount: Int): Int {
        return when (repeatType) {
            1 -> {
                // 周重复：按 week_rule 数量
                if (weekRule.isEmpty()) 7 else weekRule.split(",").size
            }
            2 -> {
                // 周目标：使用 target_count
                if (targetCount > 0) targetCount else 7
            }
            else -> 7
        }
    }
}
```

### 7.4 上一周/下一周导航

```kotlin
// 上一周
ivPrev.setOnClickListener {
    currentDate = currentDate.minusWeeks(1)
    updateWeekRange(tvWeekRange)
    loadWeekData(currentDate, recyclerView)
}

// 下一周（不能超过当前周）
ivNext.setOnClickListener {
    if (!isCurrentWeek()) {
        currentDate = currentDate.plusWeeks(1)
        updateWeekRange(tvWeekRange)
        loadWeekData(currentDate, recyclerView)
    }
}

// 判断是否为当前周
private fun isCurrentWeek(): Boolean {
    val currentWeekStart = currentDate.with(DayOfWeek.MONDAY)
    val todayWeekStart = today.with(DayOfWeek.MONDAY)
    return currentWeekStart == todayWeekStart
}

function formatDate(date) {
    const d = new Date(date);
    const year = d.getFullYear();
    const month = String(d.getMonth() + 1).padStart(2, '0');
    const day = String(d.getDate()).padStart(2, '0');
    return `${year}-${month}-${day}`;
}
```

## 九、性能考量

### 9.1 当前实现

| 查询 | 执行次数 | 时间复杂度 |
|------|---------|-----------|
| 周统计 | 2次查询 + N次遍历 | O(N) |
| 月统计 | 2次查询 + N次遍历 | O(N) |
| 年统计 | 2次查询 + N次遍历 | O(N) |

### 9.2 优化建议

**大量数据时：**
1. **添加索引**
   ```sql
   CREATE INDEX idx_record_task_date ON record(task_id, record_date);
   CREATE INDEX idx_record_date ON record(record_date);
   ```

2. **使用 task_count 表**
   - 预聚合的计数表可直接用于统计
   - 减少对 record 表的 COUNT 查询

3. **缓存**
   - 周/月统计变化不大，可缓存
   - 打卡时失效缓存
