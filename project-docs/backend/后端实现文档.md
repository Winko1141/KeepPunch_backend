```sql
-- 任务表
CREATE TABLE task (

  status INT DEFAULT 1,

  id INT AUTO_INCREMENT PRIMARY KEY,

  record_date DATE NOT NULL,

  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  UNIQUE KEY unique_task_date (task_id, record_date)

-- 分类表
CREATE TABLE category (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  icon VARCHAR(100) DEFAULT 'default',
  color VARCHAR(20) DEFAULT '#FF6B00',
  sort_order INT DEFAULT 0,
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP

-- 图标表
CREATE TABLE icon (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  display_name VARCHAR(100),
  icon_type VARCHAR(20) DEFAULT 'system',
  icon_content TEXT,
  icon_format VARCHAR(20) DEFAULT 'png',
  sort_order INT DEFAULT 0,
  is_active TINYINT DEFAULT 1,
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP

-- 任务计数表

  id INT AUTO_INCREMENT PRIMARY KEY,
```

```javascript
router.post('/create', async (ctx) => {
    const { task_name, icon, repeat_type, ... } = ctx.request.body;
```

## 四、打卡路由 (routes/record.js)

### 4.1 打卡

**接口：** `POST /record/checkin`

**请求体：**
```json
{
    "task_id": 1,
    "record_date": "2026-06-09"
}
```

**事务流程：**

```
1. 获取数据库连接
2. 开启事务
3. 检查是否已打卡（唯一约束检查）
4. 插入打卡记录
5. 更新/插入计数表
6. 提交事务
7. 归还连接
```

**核心代码：**
```javascript
const connection = await pool.getConnection();
try {
    await connection.beginTransaction();
    
    // 防重复检查
    const checkSql = `SELECT * FROM record WHERE task_id = ? AND record_date = ?`;
    const [existing] = await connection.query(checkSql, [task_id, record_date]);
    if (existing.length > 0) {
        await connection.rollback();
        ctx.body = { code: 400, msg: '今天已打卡' };
        return;
    }
    
    // 插入记录
    const insertSql = `INSERT INTO record (task_id, record_date, is_finish)
                       VALUES (?, ?, 1)`;
    const [result] = await connection.query(insertSql, [task_id, record_date]);
    
    // 更新计数（使用 ON DUPLICATE KEY UPDATE）
    const upsertWeekCountSql = `
        INSERT INTO task_count (task_id, year, month, week, total_count)
        VALUES (?, ?, ?, ?, 1)
        ON DUPLICATE KEY UPDATE total_count = total_count + 1
    `;
    await connection.query(upsertWeekCountSql, [task_id, year, month, weekOfYear]);
    
    await connection.commit();
    ctx.body = { code: 200, msg: '打卡成功', data: { id: result.insertId } };
    
} catch (err) {
    await connection.rollback();
    // 错误处理
} finally {
    connection.release();
}
```

**日期验证：**
- 格式检查：必须为有效日期
- 范围检查：不能大于今天

### 5.4 任务完成率

**接口：** `GET /stats/rate/:taskId`

**核心算法：**

```javascript
// 总打卡次数
const totalCountSql = `SELECT COUNT(*) as count FROM record WHERE task_id = ?`;

// 获取所有打卡日期
const allRecordsSql = `SELECT record_date FROM record 
                       WHERE task_id = ? ORDER BY record_date ASC`;

// 计算连续天数
let currentStreak = 0;
let maxStreak = 0;

// 检查是否从今天/昨天开始连续
const lastDate = sortedDates[sortedDates.length - 1];
const daysDiff = Math.floor((today - lastDate) / (1000 * 60 * 60 * 24));

if (daysDiff <= 1) {
    // 从最后日期往前数连续
    for (let i = sortedDates.length - 2; i >= 0; i--) {
        const diff = Math.floor((sortedDates[i + 1] - sortedDates[i]) / (1000 * 60 * 60 * 24));
        if (diff === 1) {
            currentStreak++;
        } else {
            break;
        }
    }
}

// 计算历史最长连续
let tempStreak = 1;
for (let i = 1; i < sortedDates.length; i++) {
    const diff = Math.floor((sortedDates[i] - sortedDates[i - 1]) / (1000 * 60 * 60 * 24));
    if (diff === 1) {
        tempStreak++;
    } else {
        maxStreak = Math.max(maxStreak, tempStreak);
        tempStreak = 1;
    }
}
maxStreak = Math.max(maxStreak, tempStreak);
```

```json
    }
}
```

### 6.2 分类列表

**特殊处理：** 关联查询任务数量

```sql
SELECT c.id, c.name, ..., COUNT(t.id) as task_count
FROM category c
LEFT JOIN task t ON c.id = t.category_id AND t.status = 1
GROUP BY c.id
ORDER BY c.sort_order ASC, c.id ASC
```
